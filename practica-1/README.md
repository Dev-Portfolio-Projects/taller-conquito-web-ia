# Calculadora de Propinas - Práctica 1

## Descripción

Este proyecto fue desarrollado en JavaScript para ejecutarse desde la consola utilizando Node.js.

El archivo **calculadora.js** solicita al usuario el monto de una cuenta y calcula automáticamente el valor de la propina correspondiente al 10%, 15% y 20%.

Además, el proyecto incluye un segundo programa (**iva.js**) que calcula el IVA del 15% de una compra y muestra el total a pagar.

## Requisitos

* Node.js
* npm
* Dependencia `prompt-sync`

## Instalación

Instalar las dependencias con el siguiente comando:

```bash
npm install
```

## Ejecución

Para ejecutar la calculadora de propinas:

```bash
node calculadora.js
```

Para ejecutar el cálculo del IVA:

```bash
node iva.js
```

## Tecnologías utilizadas

* JavaScript
* Node.js
* prompt-sync
